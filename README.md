# Bhutan Movie App (Expo)

This is a complete Expo React Native project with:

- Home (Trending banner + Popular list)
- Search
- Details (with YouTube trailer via WebView)
- Profile (watchlist persisted with AsyncStorage)
- React Navigation (Bottom Tabs + Stack)
- Expo-compatible libraries

## Setup

1. Put your TMDB v3 API key into `constants/api.js`
2. Install dependencies:
   npm install
3. Start:
   npx expo start -c
